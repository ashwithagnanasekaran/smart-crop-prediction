# 📊 Data Dictionary - Smart Crop Recommendation Dataset

This document describes the features used in the crop recommendation system.

| Column Name | Description |
|------------|-------------|
| N | Nitrogen content in the soil (kg/ha) |
| P | Phosphorus content in the soil (kg/ha) |
| K | Potassium content in the soil (kg/ha) |
| temperature | Temperature in degree Celsius |
| humidity | Relative humidity (%) |
| ph | Soil pH value (acidity/alkalinity) |
| rainfall | Rainfall in millimeters (mm) |
| label | Crop type recommended (e.g., Rice, Maize, Cotton) |

---

## 🧠 Notes
- N, P, K are primary soil nutrients essential for plant growth.
- pH indicates soil acidity or alkalinity.
- Temperature, humidity, and rainfall influence crop suitability.
- The 'label' column is the target variable used for prediction.

---

## 🎯 Purpose
This dataset is used to predict the most suitable crop based on soil nutrients and environmental conditions.
